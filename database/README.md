# WillyGuru Colour Database

Welcome to the WillyGuru Colour Database.

This folder will contain the verified Kubrow colour reference data used by the WillyGuru Colour Scanner.

## Goals

- Build the most accurate Kubrow colour database.
- Verify all 24 natural Kubrow colours.
- Store multiple verified examples of every colour.
- Improve scanner accuracy over time.
